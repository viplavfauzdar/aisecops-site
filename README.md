# AISecOps (aisecops.net) — V3
Astro + Tailwind site with SEO, GA4, Mermaid, blog+RSS, whitepaper PDF download, Netlify Forms.

## Start
```bash
npm install
npm run dev
```

## Google Analytics (GA4)
Set Netlify env var:
- `PUBLIC_GA_ID=G-XXXXXXXXXX`

## Whitepaper
Replace: `public/whitepaper/aisecops-whitepaper.pdf`
Stable link: `/whitepaper/download`

## Mermaid
Use fenced blocks:
```mermaid
flowchart LR
  A --> B
```
